# Introdução

Este projeto consiste em um sistema IoT para monitoramento e automação de hortas utilizando ESP32. O sistema coleta dados ambientais através de sensores e aciona atuadores automaticamente para manter condições ideais para o cultivo.

## Problema

Manter uma horta saudável requer atenção constante às condições ambientais como temperatura, umidade e luminosidade. Muitas pessoas não têm tempo ou conhecimento para monitorar esses fatores regularmente, resultando em plantas mal cuidadas.

## Objetivos

**Objetivo Geral:** Desenvolver um sistema automatizado de monitoramento e controle de hortas usando IoT.

**Objetivos Específicos:**

- Coletar dados de sensores (temperatura, umidade do ar/solo, luminosidade) e transmiti-los via MQTT
- Acionar atuadores (ventoinha, bomba d'água, luz) automaticamente com base nas leituras
- Armazenar histórico de leituras em banco de dados para análise

## Público-Alvo

Pessoas que cultivam hortas em casa ou pequenos produtores que desejam automatizar o cuidado com suas plantas, independente do nível de conhecimento técnico.
