# Conclusão

O projeto IoT Garden foi desenvolvido com sucesso, implementando um sistema completo de monitoramento e automação de hortas. O sistema é capaz de:

- Coletar dados de temperatura, umidade do ar, umidade do solo e luminosidade
- Transmitir dados via MQTT para armazenamento em banco de dados
- Acionar automaticamente atuadores (ventoinha, irrigação, iluminação) baseado nas condições ambientais
- Exibir informações em tempo real no display OLED
- Permitir controle manual via aplicativo mobile
